## 🧠 Human Action Recognition with CNN
Командный проект для **Яндекс.Лицея** по разработке модели свёрточной нейронной сети (CNN) для классификации действий человека на изображениях.  
Цель — создать надёжный классификатор, способный распознавать различные действия (например, бег, прыжки, сидение) по статичным кадрам.

---

## 📌 О проекте

Задача компьютерного зрения — автоматическое определение действия человека по фотографии. Такая технология может применяться в системах видеонаблюдения, спортивной аналитике, анализе поведения и робототехнике.

В проекте:
- реализован полный цикл: загрузка данных, аугментация, обучение, валидация;
- протестированы несколько архитектур CNN;
- настроены логирование и визуализация метрик.

---

## 🛠 Стек технологий

- **Язык:** Python 3.9

## 📊 Данные
Данные представляют собой изображения людей, выполняющих различные действия.  
Предобработка включает ресайз, нормализацию и аугментацию (повороты, отражения, изменение яркости) для увеличения обобщающей способности модели.

## 🚀 Быстрый старт

### 1. Клонирование и установка

```bash
git clone https://github.com/yourusername/human-action-recognition.git
cd human-action-recognition
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
pip install -r requirements.txt
```

### 2. Скачивание данных

Скачайте датасет с Kaggle и распакуйте в папку `data/` (или укажите свой путь в `config.py`).

### 3. Настройка конфигурации

Откройте `config.py` и при необходимости измените:
- `PATH_TO_DATA` — путь к данным;
- гиперпараметры (batch size, learning rate, число эпох);
- выбор модели (например, `'SimpleCNN'`, `'ResNet18'`).

### 4. Запуск обучения

```bash
python src/train.py
```

Логи и графики сохранятся в папку `logs/`.
